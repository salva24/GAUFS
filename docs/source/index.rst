Overview
--------

GAUFS provides:

- **Gaufs**: main class
- **DataGenerator**: synthetic data generators
- **read_unlabeled_data_csv / read_labeled_data_csv**: CSV loaders
- **clustering_experiments**: base & concrete clustering classes
- **evaluation_metrics**: internal & external metrics
